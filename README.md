# Torrent Searcher RoBot.
Search torrents from within telegram.

## Deploying
Get `BOT_TOKEN` from [@BotFather](https://t.me/BotFather) and `TOR_KEY` from [@Tor_Key_Bot](https://t.me/Tor_Key_Bot).
